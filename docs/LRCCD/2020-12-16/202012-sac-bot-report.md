Below is a list of highlighted activities, accomplishments and events at each of the colleges within the Los Rios
District. This report is to be used as a guide for the Student Advisory Committee representative and the
Student Trustee in providing the monthly LRCCD student report at the LRCCD Board of Trustee (BOT)
meetings. Only the information on this report will be used. Not all areas need to be completed; rather, this is a
guide on topics to consider. Approved/signed college reports are due to the Student Trustee one week prior to
each scheduled BOT meeting. The Student Trustee in consultation with the SAC Advisor will compile these
submissions for one comprehensive report which is due to District Advisor Dr. Rosario, the designated student
speaker, and SAC Advisor by noon the Friday before the BOT meeting. Late submissions/unauthorized reports
will not be presented at the BOT meeting.
 Student led-activities (i.e. March in March, campus events, fundraisers):
      ● ARC: ARC appointed a new Director of Finance to the Student Senate.
      ● CRC: CRC held a Stress Relief event from 12/8-12/10.
      ● FLC: FLC planned a De-Stress Event and Student Recognition event.
      ● SCC: SCC is compiling a Basic Needs Referral Resource for students, and held a virtual DeStress
           Fest.

 Senate and CAEB Accomplishments (i.e. surveys, engagement in campus and regional governance):
    ● ARC: ARC is planning a Spring Virtual Club Day, working to improve the structure of the ARC ASB
        website, approved a list of companies to send donation requests for an ASB scholarship, and
        approved a resolution for COVID-19 Block Grant Advocacy.
    ● CRC: CRC allocated $15,000 to purchase media equipment to increase advocacy awareness. CRC
        is working to amend their ASCRC Bylaws and Constitution, create a Transcript Notation Resolution,
        create a Block Grant Resolution for basic needs, and is working to create a Canvas course to
        increase student engagement.
    ● FLC: FLC passed the DEI and COVID-19 Block Grant, presented the new ASFLC website, updated
        the CAEB bylaws, and is involved in planning a Student Success Program for DI students.
    ● SCC: SCC registered 400 new voters through the Ballot Bowl Competition, and completed updating
        their bylaws for both Clubs & Events Board and Student Senate..
